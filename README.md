# docker-playbook

このシナリオはdockerを初期導入するための  
シナリオである。

------------------------------------------------------------
## (対応OS)
* CentOS 6.x x86_64

------------------------------------------------------------
## (設定内容)
* dockerパッケージのインストール
* サービスの自動起動
* 一般ユーザで使用できるように、UNIXソケットではなくTCPソケットで
  dockerと通信できるように設定を変更
